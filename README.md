# OSINT Toolkit

Coleção de ferramentas de OSINT (Open Source Intelligence) desenvolvidas em Python para fins educacionais.

### 🛠️ Ferramentas incluídas
- Consulta de IP (geolocalização, provedor e ASN)
- Username checker em várias redes sociais
- Verificação de e-mail + breach check
- Lookup básico de números de telefone
- Web scraper simples com suporte a proxies
- Gerador de dorks para Google

### 🛠️ Tecnologias utilizadas
- **Python 3.11**
- `requests`, `beautifulsoup4`, `colorama`
- `fake-useragent`, `python-dotenv`
- Suporte a configuração via arquivo `.env`

### Como usar

```bash
git clone https://github.com/vitorhazov/osint-toolkit.git
cd osint-toolkit
pip install -r requirements.txt
python main.py
